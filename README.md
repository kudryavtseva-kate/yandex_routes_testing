# yandex_routes_testing

## О проекте

В рамках проекта проведено тестирование веб-приложения. Целью было проверить функциональность, корректность работы форм и бизнес-логики. Были применены функциональное, негативное, регрессионное и smoke-тестирование, а также кроссбраузерная проверка.

## Используемые инструменты
- DevTools
- Система баг-трекинга
- Браузеры для кроссбраузерного тестирования (Chrome, Yandex, Firefox)

## Результаты
- Составлены тест-кейсы и чек-листы
- Проведено тестирование всех функциональных сценариев
- Найдены и оформлены баги
